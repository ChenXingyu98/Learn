# 自如对房东的套路
以下的套路，我已经亲身经历了1-3，朋友正处于4的阶段，基本可以确认1-4的真实性。5、6、7是负面新闻但尚未证实(需要等下一个付款周期)
1. 自如不赚钱，房东您能不能降价，本次已经为您申请了最大的降价折扣是几百元（套路1）。
2. 如果您不同意降价，您可以申请与自如解约，自如是正规公司一切都按照合同约（套路2）。
3. 如果您不同意降价也不申请解约，自如会发起解约，按照合同未到期的装修折损您需要赔偿我们x万元（套路3）。
4. 定期轮番电话轰炸，重复1-3(套路4)。
5. 停止打款，等房东熬不住提出解约(套路5)。
6. 停止打款的一段时间后(1-2个月)，如果房东不提出解约，自如主动发起解约(套路6)。
7. 与6同步进行，自如发送律师函要求单方解约并要求赔偿装修折旧费。

# 合同的法律分析
综上，把套路话术和合同条款对照起来看，自如的思路我们就清楚了。以下这些可能性是我们的分析和揣测，不代表自如真的打算这么做，但从法律的角度任何可能性我们都要尽量考虑到，任何风险我们都要尽量规避掉。
1. 同意降价并“申请折扣降价”，走《中华人民共和国合同法》 第七十七条，业主被自愿降价。
2. 不同意降价但主动申请解约，按照第6.3条+第3.3条，自如不用赔钱，房东要赔偿装修设施折折损。
3. 不同意降价自如将发起单方解约，按照第7.3条+第3.3条，自如需要支付2个月赔偿金，通过第3.3条要求房东赔偿装修设施折折损。
4. 通过反复洗脑能按照1-3达成是最好的，如果无法达成就提出自如不赔偿2个月，房东不用赔偿装修设施折折损。
5. 停止打款，这里有两种可能，第一种可能是房东有经济压力熬不住且没注意到合同条款主动申请解约，按照第6.3条+第3.3条，自如不用赔钱，房东要赔偿装修设施折折损。第二种可能是房东不理智对房子或资产使用人采取了某些行动，自如可以以第6.2条 (1)b条或第6.2条 (2)d条主张第7.2条甲方违约，甲方需要赔偿2个月赔偿金+装修设施折折损。
6. 自如单方解约有两种可能性。第一种可能是按照第7.3条违约，需要支付2个月赔偿金，但该条违约没有约定自如违约装修设施折损，然后通过第3.3条要求房东赔偿。第二种可能是先按照合同法第九十四条不可抗力提出解约要求，如果没获得支持再按照合同法第一百一十条提出解约必然可以获得支持，在支付2个月赔偿金后继续通过第3.3条要求房东赔偿。
7. 发送律师函的目的是考虑了大部分妥协难以承受打官司的成本，尽可能逼房东妥协。

# 相关的法律法规和司法解释
在讨论房东应对方案前，我们先把与房东息息相关相关的法律法规和司法解释整理如下，对法律不感冒的朋友可以跳过直接看房东应对方案：
1. 根据《中华人民共和国合同法》 第三条关于“平等原则  合同当事人的法律地位平等，一方不得将自己的意志强加给另一方。”
2. 根据《中华人民共和国合同法》 第三十九条关于“格式合同条款定义及使用人义务 采用格式条款订立合同的，提供格式条款的一方应当遵循公平原则确定当事人之间的权利和义务，并采取合理的方式提请对方注意免除或者限制其责任的条款，按照对方的要求，对该条款予以说明。 格式条款是当事人为了重复使用而预先拟定，并在订立合同时未与对方协商的条款。”
3. 根据《中华人民共和国合同法》 第四十条关于“格式合同条款的无效 格式条款具有本法第五十二条和第五十三条规定情形的，或者提供格式条款一方免除其责任、加重对方责任、排除对方主要权利的，该条款无效。”
4. 根据《中华人民共和国合同法》 第四十一条关于“格式合同的解释 对格式条款的理解发生争议的，应当按照通常理解予以解释。对格式条款有两种以上解释的，应当作出不利于提供格式条款一方的解释。格式条款和非格式条款不一致的，应当采用非格式条款。”

5. 根据《中华人民共和国合同法》 第七十七条关于 “ 合同变更条件 当事人协商一致，可以变更合同。 法律、行政法规规定变更合同应当办理批准、登记等手续的，依照其规定。”
6. 根据《中华人民共和国合同法》 第九十四条关于 “ 合同的法定解除 有下列情形之一的，当事人可以解除合同： （一）因不可抗力致使不能实现合同目的； （二）在履行期限届满之前，当事人一方明确表示或者以自己的行为表明不履行主要债务； （三）当事人一方迟延履行主要债务，经催告后在合理期限内仍未履行； （四）当事人一方迟延履行债务或者有其他违约行为致使不能实现合同目的； （五）法律规定的其他情形。”
7. 根据《中华人民共和国合同法》 第九十七条关于 “解除的效力 合同解除后，尚未履行的，终止履行；已经履行的，根据履行情况和合同性质，当事人可以要求恢复原状、采取其他补救措施，并有权要求赔偿损失。”
8. 根据《中华人民共和国合同法》 第一百零八条关于 “拒绝履行 当事人一方明确表示或者以自己的行为表明不履行合同义务的，对方可以在履行期限届满之前要求其承担违约责任。”
9. 根据《中华人民共和国合同法》 第一百一十条关于 “非金钱债务的违约责任 当事人一方不履行非金钱债务或者履行非金钱债务不符合约定的，对方可以要求履行，但有下列情形之一的除外： （一）法律上或者事实上不能履行； （二）债务的标的不适于强制履行或者履行费用过高； （三）债权人在合理期限内未要求履行。”
10. 根据《中华人民共和国合同法》 第一百一十三条关于 “损害赔偿的范围 当事人一方不履行合同义务或者履行合同义务不符合约定，给对方造成损失的，损失赔偿额应当相当于因违约所造成的损失”的规定，因合同一方存在违约，合同另一方为此 聘请律师提供专业法律服务而支出的合理费用，属于违约方违约给守约方造成的损失，应由其承担。
11. 根据《中华人民共和国合同法》 第一百一十七条关于 “不可抗力 因不可抗力不能履行合同的，根据不可抗力的影响，部分或者全部免除责任，但法律另有规定的除外。当事人迟延履行后发生不可抗力的，不能免除责任。 本法所称不可抗力，是指不能预见、不能避免并不能克服的客观情况。”
12. 根据《城镇房屋租赁合同解释》 第十条关于“承租人经出租人同意装饰装修，租赁期间届满或者合同解除时，除当事人另有约定外，未形成附合的装饰装修物，可由承租人拆除。因拆除造成房屋毁损的，承租人应当恢复原状。”
13. 根据《城镇房屋租赁合同解释》 第十条关于“第十一条承租人经出租人同意装饰装修，合同解除时，双方对已形成附合的装饰装修物的处理没有约定的，人民法院按照下列情形分别处理：(一)因出租人违约导致合同解除，承租人请求出租人赔偿剩余租赁期内装饰装修残值损失的，应予支持;(二)因承租人违约导致合同解除，承租人请求出租人赔偿剩余租赁期内装饰装修残值损失的，不予支持。但出租人同意利用的，应在利用价值范围内予以适当补偿;(三)因双方违约导致合同解除，剩余租赁期内的装饰装修残值损失，由双方根据各自的过错承担相应的责任;(四)因不可归责于双方的事由导致合同解除的，剩余租赁期内的装饰装修残值损失;”

# 房东的应对方案
写在应对方案之前，我们希望阐述一下我们的观点，阐述一下撰写并发布方案的目的：
1. 链家和自如必须直面自己商业模式的问题并承当相应的损失，不应该把损失和风险转嫁给房东和租客。融资风光的时候没共富贵，患难就想到分摊风险了。
2. 支持链家和自如因为经营不善而单方解约并承当相应的损失，不应该用格式合同条款霸王条约套路业主，更不该用利用大部分房东不懂法或无法承受打官司的压力把损失转嫁给业主。
3. 链家和自如应该以合法的方式进行解约并赔偿损失，不应该利用部分房东在法律和精力上的不对等，压榨房东。
4. 房东也必须意识到，房东和租客都是链家和自如的受害者，房东切忌因为一时冲动对租客或资产采取过激行动，一方面不利于社会稳定，一方面因第7.2条甲方违约陷入不利境地。房东和租客要联合起来，一起向自如讨公道，伟大的毛主席说过，要把我们的人搞得多多的,把敌人的人搞得少少的。
5. 要相信我们的国家是一个法制国家，虽然自如钻了一些法律空子，如果没有人站出来用法律的形式去抗争，就会有更多不懂法律的受害者出现。希望这篇应对方案能让更多的自如房东站出来，拿起法律的武器捍卫自己的权利。

从小喜欢看古龙的小说，大战之前各种铺垫各种烘托各种造势，但到了真正的大战却是寥寥几笔带过。前面洋洋洒洒铺垫了这么多，一方面是把法律法规都整理出来，另一方面也是理顺我们的思路。到房东应对方案的阶段，就可以简明扼要执笔。

在看应对方案之前，房东应该先根据自己的情况确定自己的诉求并选择相应的法律应对方案：
1. 愿意降价不愿意折腾的房东
2. 不愿意降价也不愿意折腾的房东
3. 不愿意降价愿意折腾的房东

在这里我们要重申一下，从法律的角度任何可能性我们都要尽量考虑到，任何风险我们都要尽量规避掉，应对方案不是万能的，但没有应对方案是万万不能。

愿意降价不愿意折腾的房东的应对方案如下：
1. 同意并通过补充协议降价
2. 补充协议必须约定自如多少时间内不能再要求降价
3. 房东保留后续起诉的权利

不愿意降价房东的应对方案如下：
0. 接到自如管家来电时，友善、微笑地提醒自如管家，您以下通话讲被录音。
1. 自如不赚钱，房东您能不能降价，本次已经为您申请了最大的降价折扣是几百元（套路1）。答复：我们不同意降价，请自如自觉自动履行合同。
2. 如果您不同意降价，您可以申请与自如解约，自如是正规公司一切都按照合同约（套路2）。答复：我们不会主动申请解约，我们期待自如能够自觉自动履行合同。如果自如想要单方解约请按照法律法规合法解除。
3. 如果您不同意降价也不申请解约，自如会发起解约，按照合同未到期的装修折损您需要赔偿我们x万元（套路3）。答复：我们支持自如按照法律法规单方解除合同，根据第7.3条自如需要赔偿两个月租金。关于装修设施折旧我们不支持，理由有三：
    1. 第7.2条和第7.3不对等，在签约时自如表示这是格式合同条款不能更改，7.3对自如单方违约的装修折损条款避而不谈。根据合同法第三、三十九、四十、四十一条，当自如单方违约时第3.3条不能作为自如索赔依据；
    2. 根据合同法第一百一十三条，自如单方违约时需要损失赔偿额应当相当于因违约所造成的损失，其中应该包括了装修设施折损
    3. 房东和自如的合同是资管合同，目前尚无对应的司法解释，但根据《城镇房屋租赁合同解释》 第十条，当自如单方违约时第3.3条不应该作为自如索赔的依据。
4. 定期轮番电话轰炸，重复1-3(套路4)。答复：我白天要上班，晚上要陪家人，请在17:00-20:00这段时间联系协商，其他时间请勿影响工作和休息。
5. 停止打款，等房东熬不住提出解约(套路5)。到了这一步，房东应根据自己的情况选择应对方案：
    1. 无法等待的房东，在这个阶段尽量跟自如谈两清，自如不赔偿违约金，也放弃装修设施折损的主张。走这个方案的房东，一定要自如公开承诺放弃装修设施主张后再发起解约，避免被二次套路。
    2. 可以等待并希望追回损失的房东，要明确告知自如我们将以维权群组织集体诉讼法庭见，并有情提醒自如根据《中华人民共和国合同法》 第一百一十三条 如果作为违约方败诉，自如还需要承担业主的律师费用支出。
    3. 不管走方案1还是方案2，在自如停止打款的一个月内都不要对房子和租客有过激行动，避免落自如口实以合同第7.3条解约。
6. 停止打款的一段时间后(1-2个月)，如果房东不提出解约，自如主动发起解约(套路6)。应对：
    1. 合同约定付款日15天后没有收到钱，根据第6.2条 违约解除权 (2)a条自如已经构成违约，房东应该主动给自如沟通联系并告知其已经违反了6.2 (2) a。
    2. 在1的基础上，15天后如果还没有收到房租，房东应该主动给自如沟通联系并告知其已经违反了6.2 (2) b。
    3. 无法等待的房东，在这个阶段尽量跟自如谈两清，自如不赔偿违约金，也放弃装修设施折损的主张。走这个方案的房东，一定要自如公开承诺放弃装修设施主张后再发起解约，避免被二次套路。
    4. 可以等待的房东，组建维权群准备计提诉讼，通过媒体（扩大影响）、12345市长热线（寻找主管的市级单位）等多渠道沟通反馈，集资聘请专业律师准备应诉。根据《中华人民共和国合同法》 第一百一十三条 如果作为违约方败诉，自如还需要承担业主的律师费用支出，有法律圈资源的房东也可以联系公益诉讼（参见徐斌律师 蛋壳租客vs微众银行 公益集体诉讼案）。
7. 与6同步进行，自如发送律师函要求单方解约并要求赔偿装修折旧费。应诉要点：
    1. 依据自如的《资产管理服务合同》的约定，房东将房屋交付给自如公司使用，自如公司向业主支付保证金并定期支付固定金额的预期合理收益，业主同意自如公司将涉诉房屋进行装修配置后转租他人，由他人与自如公司签订房屋租赁合同并直接向自如公司支付租金，且涉诉房屋是否转租，均不影响自如公司向业主支付预期合理收入。因此，《资产管理服务合同》属房屋租赁合同纠纷，应适用合同法中租赁合同部分，并按不动产纠纷确定管辖。
    2. 作为房东不管是电话、当面还是法庭，我们要一再强调，希望链家和自如作为行业老大，应当全面严格履行资产管理合同。根据《中华人民共和国合同法》 第一百一十七条，主张不可抗力的一方除了证明不可抗力事实发生，还需要证明该不可抗力和合同的不能履行之间有因果关系。疫情属于不可抗力因素，但是并不必然导致对方合同不能履行。长租公寓目前面临的合同履行后的利润等问题，应该认定为一般的商业风险。商业主体在履约过程中不敢担当、惧怕商业风险，并转嫁风险的做法，属于滥用权利、逃避义务。
    3. 作为房东不管是电话、当面还是法庭，我们要一再强调，如果链家和自如坚决不履行合同，我们认可链家和自如可依法根据《中华人民共和国合同法》第九十四条发起单方违约解约的权利，但我们也敦促链家和自如要依照《中华人民共和国合同法》 第九十七条、第一百零八条、第一百一十条、第一百一十三条依法履行赔偿义务。我们主张的赔偿如下：
        1. 首年空置期是按照合同完整执行周期计算，因自如单方违约，需要重新计算首年空置期并赔偿（大概率得到法院支持）。
        2. 本年空置期是按照本年完整执行周期计算，因自如单方违约，需要重新计算本年空置期并赔偿（大概率等得不到法院支持）。
        3. 根据合同约定的两个月预期收入赔偿(肯定得到法院支持)。
        4. 自如单方违约造成的误工费。
        5. 因自如违约要求单方解除并通过格式条款对房东进行威胁，房东为此聘请律师提供专业法律服务而支出的合理费用，属于违约方违约给守约方造成的损失，应由其承担。
    4. 作为房东不管是电话、当面还是法庭，合同约定的违约责任不等对，并利用藏在违约责任以外的合同第3.3条对单方进行免责，第3.3条对自如单方解约免责部分并没有以加粗、黑体字、下划线、标注进行说明，没有尽到合同条款说明义务，合同也没有留设补充条款供各方对合同进行修改、增补、删减，事后以守约方“自愿”签订的为由，主张对守约方不利的条款内容，应认定为利用格式合同条款一方免除其责任、加重守约方责任、排除守约方主要权利的，自如作为单方违约的一方，该条款的权利主张是无效。
